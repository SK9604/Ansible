# Windows Server 2016

- 윈도우 서버 2016(Windows Server 2016)는 마이크로소프트의 서버 계열의 운영 체제이다.
- 정식 버전은 2016년 10월 12일(한국시간 13일 새벽 2시)에 출시하였다.
- 윈도우10의 서버 버전이다.(커널이 같음)

## Windows 운영체제 변천사

Windows 95는 가장 혁신적인 인터페이스로 출시되었고 최신 윈도우 계열 운영체제의 인터페이스도 이와 비슷하다.
